# RealSTEM

RealSTEM is a news-to-lesson AI concept for teachers. The initial scaffold is a polished landing page that frames the product, explains the problem, and outlines the first workflow.

## Stack

- Next.js 15
- React 19
- TypeScript
- App Router

## Run locally

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Open `http://localhost:3000`

## What is included

- A branded, responsive landing page
- A product narrative for the news-to-lesson workflow
- A starter project setup with scripts and TypeScript configuration
